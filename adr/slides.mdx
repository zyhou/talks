# Architectural Decision Records (ADR)

**"An architecture decision record is a <span style="color: red">short text file</span> in a format to an Alexiandran pattern that describes a <span style="color: red">set to forces</span> and a <span style="color: red">single decision</span> in a response to those forces."** - Proposé part Michael Nygard

---

# C'est quoi un ADR ?

<ul>
    <li>Légé basé sur du texte</li>
    <li>Enregistré avec le projet</li>
    <li>Une décision par fichier (0001-git-for-version-control.md)</li>
</ul>
