# Architectural Decision Records (ADR)

__"An architecture decision record is a **short text file** in a format to an Alexiandran pattern that describes a **set to forces** and a **single decision** in a response to those forces."__ -Proposé part Michael Nygard

---

# C'est quoi un ADR ?

<ul>
    <li>Léger basée sur du texte</li>
    <li>Enregistré avec le projet</li>
    <li>Une décision par fichier (0001-git-for-version-control.md)</li>
</ul>
